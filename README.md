## Get Started

First, install dependencies:

```bash
npm run install

```
Second, create .env file:

```bash
DATABASE_URL="mongo_url"

```

Third, run the development server:

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
